# **App Name**: AIESEC Nepal Portal

## Core Features:

- Global Exchange Navigator: Interactive cards for Global Volunteer, Talent, and Teacher programs consuming the /api/programs endpoint.
- AI Exchange Matcher Tool: A smart tool that uses generative reasoning to suggest specific exchange programs based on a user's skills and interests gathered via the form.
- Dynamic Leadership Form: A shared submission page that uses query parameters to dynamically generate fields for members, exchanges, or partners via the /api/submit endpoint.
- Local Committee Visualizer: A scalable grid system that fetches and displays real-time statistics and years of establishment for all local committees via /api/lcs/stats.
- Filtered Content Engine: A blog and events hub that allows users to browse and filter organizational updates by categories like 'Exchange' and 'Impact' using REST API endpoints.
- Testimonial Trust Carousel: A sleek, responsive carousel featuring authenticated testimonials from AIESEC members and partners fetched from /api/testimonials.
- Partner Ecosystem Showcase: Categorized display of current and previous organizational partners with automated logo rendering and contribution typing.

## Style Guidelines:

- Primary Brand Color: High-energy AIESEC Red (#D21034) representing leadership and passion.
- Secondary Brand Palette: AIESEC Blue (#037EF3), Teal (#0CB9C1), Orange (#F48924), Coral (#F85A40), and Purple (#7552CC) for program differentiation.
- UI Elements and Accents: Vibrant Green (#00c16e), Yellow (#ffc845), Grey (#52565E), and Light Grey (#f5f5f5) for status and interactions.
- Background: A subtle, desaturated off-white (#F9F5F5) to maintain a clean and professional aesthetic.
- Typography pairing: 'Inter' for objective body text clarity and 'Poppins' for headlines to deliver a geometric, contemporary avant-garde vibe.
- Minimalist stroke icons from Lucide React to maintain a clean, high-end agency look across all seven pages.
- Responsive editorial layout with clear hierarchical spacing and a featured hero section utilizing a 12-column grid.
- Subtle entrance staggers and smooth state transitions using Framer Motion to mirror the fluid energy of youth leadership.
